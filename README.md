# Polish Music Dataset

# About project:

Polish Music Dataset is the largest dataset with information about artists, songs and lyrics in Poland (now only Hip Hop artists).  
Data are from Genius and YouTube (not yet).  

# Data:

## Data types:
  ```title``` - object  
  ```album``` - object  
  ```year``` - object  
  ```lyrics``` - object  
  ```image``` - object  
  ```artist``` - object
